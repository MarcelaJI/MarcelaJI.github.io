## Reconocimiento de imágenes 

El reconocimiento de imágenes pueden ser utilizadas para obtener información valiosa sobre personas y los lugares

Una herramienta útil para este reconocimiento es:

[PinEyes](https://pimeyes.com/en)

El funcionamiento de esta herramienta se basa en el análisis de patrones faciales, que son comparados con una base de datos de imágenes en línea para encontrar similitudes.
