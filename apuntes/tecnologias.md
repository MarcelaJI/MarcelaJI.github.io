## Identificación de las tecnologías de una página web

La primera herramienta para identificar tecnologías en una página web es *whatweb* nos va a permitir detectar las tecnologías que corren detrás de una página web y en ocasiones el gestor de contenidos.

Se ejecuta así:

```bash
whatweb https://miwifi.com
```

La siguiente herramienta es *Wappalyzer* es una extensión del navegador que detecta y muestra las tecnologías utilizadas en la página web.

La siguiente herramienta es *Builtwith.com* es una herramienta en línea de comandos que nos permite identificar las tecnologías utilizadas en una página web.
